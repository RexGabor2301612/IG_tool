About

S&R Facebook Extract

What this tool does

This workspace helps collect visible public Facebook post information into Excel using a guided browser session, structured logs, and clear extraction status.

Collection focus

The scraper can record Facebook post links, detected dates, visible reactions, comments count, share count, and optional visible comment samples when they are publicly accessible.

Manual login workflow

If Facebook requires login, the system opens a real browser window and pauses. You complete login there, then return here and click GO when the page is ready.

Accuracy first

The extractor leaves unavailable values blank instead of guessing, and writes a diagnostic workbook when no rows match the selected date coverage.
